# Standard Operating Procedure (SOP)

## Daily Workflow
1. Morning: `/plan`
2. Active work: `/project` (or `/dump` when context is messy)
3. Overwhelm: `/freeze`
4. End of day: `/wrap`

## Command Definitions
For exact command behavior, always use:
- `03_AI_Brain_&_Prompts/06_Slash_Command_Library.md`

## Deprecated Inputs
- `/intake` → `/plan`
- `/next` → `/project`
- `/stuck` → `/freeze`
