# Quick Reference Cheat Sheet

## Active Commands (Final Contract)
1. `/plan` — Inbox + calendar scan, propose 2–3 daily execution blocks.
2. `/project` — Micro-step a project and generate first draft.
3. `/dump` — Convert unstructured notes into an execution plan.
4. `/freeze` — Immediate rescue protocol and pressure reset.
5. `/wrap` — End-of-day summary and defer/reschedule workflow.

## Required Approval Rule
No external write actions occur without explicit approval.

## Deprecated Commands
- `/intake` → use `/plan`
- `/next` → use `/project`
- `/stuck` → use `/freeze`
